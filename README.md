# Party Games

A collection of party games. Each subfolder contains a different game with its own rules and instructions.

## Games

Browse the subfolders to find a game to play!

## Contributing

To add a new game, create a subfolder with the game's name and include a `README.md` with:

- **Name** of the game
- **Players** — number of players required
- **Materials** — any materials needed
- **Rules** — how to play
